# kserve-crd-minimal

Helm chart for deploying minimal kserve crds without validation

![Version: v0.15.2](https://img.shields.io/badge/Version-v0.15.2-informational?style=flat-square)

## Installing the Chart

To install the chart, run the following:

```console
$ helm install kserve-crd-minimal oci://ghcr.io/kserve/charts/kserve-crd-minimal --version v0.15.2
```
